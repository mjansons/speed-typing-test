## Typing Test Application

### Features

- **60 Second Timer**
  - Default setting with lowercase letters and no special characters.
  
- **Test Controls**
  - Press `Enter` to restart the same test.
  - Press `Esc` to get a brand new text.

- **Adjustable Settings**
  - Option to include additional buttons/triggers to adjust test duration.
  - Option to include additional characters (uppercase, lowercase, numbers, commas, etc.).

- **Dynamic Word Management**
  - Words dynamically removed from the top row, also while re-sizing window.
  - Asynchronous fetching of new words to replace removed ones.

- **Typing Constraints**
  - Not able to go back after a correctly typed word.

- **Local Storage Metrics**
  - Metrics are stored in local storage for persistence.

- **Performance Comparison**
  - Provides judgement about your current test in comparison to the previous test after each test.

- **Statistics Display**
  - Comprehensive stats are displayed after each test.

- **Caret Navigation**
  - Includes a caret/cursor for navigation.

- **Character Handling**
  - Excess characters are also added to the text.
